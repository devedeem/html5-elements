Headings (Block Element)

Note:
Heading tags are used to create headings h1 - h6 Headings have a default size of 24px and can be changed using the font tag.
The heading elements will always appear in order from largest to smallest (i.e., H1 is larger than H3).

<h1>This is heading 1</h1>

<h2>This is heading 2</h2>

<h3>This is heading 3</h3>

<h4>This is heading 4</h4>

<h5>This is heading 5</h5>
<h6>This is heading 6</h6>

Paragraph (Block Element)

<p>This is a paragraph</p>

HTML anchor tag (Inline Element)

<a href="https://www.example.com">This is a link</a>

The href attribute in HTML is used in anchor () tags to specify the URL of the page that the link goes to.

HTML span tag (Inline Element)

<span>This is a span element</span>

The span tag in HTML is used to group inline-elements in a document and can be styled with CSS or manipulated with JavaScript.

HTML div tag (Block Element)

<div>This is a div block</div>

The div tag in HTML is used as a container for other HTML elements and can be styled with CSS or manipulated with JavaScript.

HTML hr tag (Block Element)

<hr>

The hr tag in HTML is used to create a thematic break between paragraph-level elements. It is most often displayed as a horizontal rule.

HTML form tag (Block Element)

<form action="/submit"><input type="text" name="name"></form>

The form tag in HTML is used to create an HTML form for user input. It can contain input elements like text fields, checkboxes, radio-buttons, submit buttons and more.

HTML label tag (Inline Element)

<label for="name">Name:</label>

The label tag in HTML is used to provide a text description for a form control. The 'for' attribute of the label tag should be equal to the id attribute of the related element to bind them together.

HTML input tags (Inline Element)

<input type="button" value="Button">

The 'button' type creates a clickable button.

<input type="checkbox" id="subscribe" name="subscribe">

The 'checkbox' type lets the user toggle a state, such as when selecting items in a list.

<input type="color" id="favcolor" name="favcolor">

The 'color' type provides a color picker for selecting a color.

<input type="date" id="birthday" name="birthday">

The 'date' type allows the user to select a date.

<input type="datetime-local" id="meeting" name="meeting">

The 'datetime-local' type allows the user to select a date and time with no time zone.

<input type="email" id="email" name="email">

The 'email' type is used for input fields that should contain an e-mail address.

<input type="file" id="file" name="file">

The 'file' type lets the user choose one or more files from their device storage.

<input type="hidden" id="userid" name="userid">

The 'hidden' type allows web developers to include data that cannot be seen or modified by users when a form is submitted.

<input type='image' src='/submit.gif' alt='Submit'>

The 'image' type defines an image as the submit button.

<input type='month' id='birthdaymonth' name='birthdaymonth'>

The 'month' type allows the user to select a month and year.

<input type='number' id='quantity' name='quantity'>

The 'number' type is used for input fields that should contain a numeric input.

<input type='password' id='password' name='password'>

The 'password' type creates a text box that obscures input to prevent sensitive data from being seen.

<input type='radio' id='radio' name='radio'>

The 'radio' type lets the user select one option from a set.

<input type='range' id='volume' name='volume'>

The 'range' type is used for input fields that should contain a value from a range of numbers.

<input type='reset'/>

The 'reset' type resets all values in the form.

<input type='search' id='googlesearch'/>

The 'search' type is used for search fields and behaves like a regular text field.

<input type='submit'/>

The 'submit' type creates a button that submits the form data to the server.

<input type='tel' id='phone_number'/>

The 'tel' type is used for input fields that should contain a telephone number.

<input type='text' id='name' name='name'/>

The 'text' type is used for single-line text input fields.

<input type='time' id='meeting_time'/>

The 'time' type allows the user to select a time (no time zone).

<input type='url' id='homepage'/>

The 'url' type is used for input fields that should contain a URL address.

<input type='week' id='week_of_year'/>

The 'week' type allows the user to select a week and year.

HTML fieldset tag (Block Element)

<fieldset></fieldset>

The fieldset tag in HTML is used to group related elements in a form. It draws a box around the related elements.

HTML legend tag (Block Element)

<legend">Personalia:</legend>

The legend tag in HTML is used to provide a caption for the fieldset element. The caption can be styled using CSS.

Here is how they can be used together 

  <form action="#"> 
   <fieldset> 
    <legend>Personalia:</legend> 
    <label for="fname">First name:</label> 
    <input type="text" id="fname" name="fname"> 
    <input type="submit" value="Submit"> 
   </fieldset> 
  </form>
